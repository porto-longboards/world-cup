Configurar o Bolão

Siga 3 passos para criar o banco de dados gratuito. Menos de 5 minutos, sem cartão.

Passo 1 — Criar conta no JSONBin
  1. Acesse jsonbin.io → Sign Up
  2. Crie conta com Google (gratuito)
  3. Clique em API Keys no menu do topo
  4. Copie a chave (começa com $2b$...)

Passo 2 — Criar o banco de dados
  1. No JSONBin, clique em New Bin
  2. Apague o texto e cole exatamente:
{"resultados":{},"palpites":{}}
  1. Clique em Create Bin
  2. Copie o BIN ID no topo
