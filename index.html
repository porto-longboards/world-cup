Configurar o Bolão

Siga 3 passos simples para criar o banco de dados gratuito. Leva menos de 5 minutos e não precisa de cartão.

Passo 1 de 3 — Criar conta no JSONBin
  1. Acesse jsonbin.io e clique em Sign Up
  2. Crie conta com Google ou e-mail (gratuito)
  3. Após entrar, clique em API Keys no menu do topo
  4. Copie a chave que aparecer (começa com $2b$...)

Passo 2 de 3 — Criar o banco de dados
  1. No JSONBin, clique em New Bin (botão verde)
  2. No campo de conteúdo, apague o texto e cole exatamente:
{"resultados":{},"palpites":{}}
  1. Clique em Create Bin
  2. No topo da página, copie o BIN ID (ex: 64a1b2c3d4e5f6...)

Passo 3 de 3 — Configurar acesso privado
  1. Ainda na página do Bin, clique no ícone de cadeado 🔒
  2. Ative a opção "Make Bin Private"
  3. Isso garante que só quem tem a API Key consegue ler os dados
✓ Configuração completa! Clique em Salvar para começar.
